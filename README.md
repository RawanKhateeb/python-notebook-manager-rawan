# Personal Notebook Manager

A simple Python console application that lets you create and manage personal notes.

Each note includes:
- **Title**
- **Content**
- **Tags**
- **Creation date**

Notes are saved in a JSON file so the data stays between sessions.

---

## Features

- Add a new note  
- List all notes  
- Search notes by keyword  
- Filter notes by tag  
- Edit a note  
- Delete a note  
- Save notes to a JSON file  
- Load notes automatically at startup  

---

## How to Run

1. Make sure Python 3 is installed.
2. Open the project folder in VS Code.
3. Run the following command in the terminal:

```bash
python main.py
